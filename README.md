アプリ・システム開発をまとめ

概要
本システムは、ユーザーがオンラインでファッションアイテムを閲覧・購入できる
ECサイト（オンラインショップ） です。
● フロントエンド（Home、Men、Women、Accessory、About Us ページ）は完成して
おり、
バックエンドは Node.js（Express）と MongoDB を使用して開発中で、ユーザー登
録・ログイン機能や注文処理機能を追加しています。
● 最終的には、ユーザーがアカウントを作成し、商品をカートに追加・購入できる完
全なECサイトを完成させることを目標としています。
システムのアピールポイント
 グリーン系（#8C907E）を基調とした 落ち着いたデザインと高級感のあるUI
 JavaScriptによる スムーズなアニメーションとホバー効果
 Node.js と MongoDB による 柔軟で高速なデータ処理
 Cookieを利用したログイン維持機能で、複数デバイスからのアクセスが可能
 Chatbotを追加
